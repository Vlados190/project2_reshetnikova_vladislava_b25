# Проект: Примитивная база данных на Python

Этот проект реализует примитивную базу данных с возможностью работы через консоль.  
Поддерживаются основные операции CRUD: создание таблиц, добавление, выборка, обновление и удаление записей, удаление таблиц.  
Используются только стандартные библиотеки Python и внешние библиотеки: `prettytable` и `prompt` (если нужно для отображения).

Особенности:
- Работа через консоль с понятным синтаксисом команд.
- Декораторы для логирования, обработки ошибок и подтверждения действий.
- Хранение данных в формате JSON.
- Полностью функциональный и тестируемый проект без использования классов.
